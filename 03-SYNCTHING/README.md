# 03-SYNCTHING

Real-time file synchronization across devices.

## Overview

Direct peer-to-peer file sync. No cloud intermediary. Bandwidth-optimized.

## Files

- `syncthing.xml` - Configuration
- `setup.md` - Initial setup procedure
- `troubleshooting.md` - Common issues

## Quick Start

Syncthing typically runs as systemd service

systemctl status syncthing
Access at http://localhost:8384

## Configuration

Device pairing via QR codes or IDs.
Selective folder sharing.

## Performance

Monitor bandwidth usage in settings.
